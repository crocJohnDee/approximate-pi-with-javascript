Estimating PI with

JavaScript

How?


JavaSript may not be great in large math calculations.
But! we can use JavaScript to visualize math in a useless, but fun-to-make way!


Below is a square canvas. When you click GO - JavaScript will add random dots to the canvas, using the .random() method.

Using the Pythagorean theorem, JavaScript can calculate the distance of the dot, from the canvas center.

If the distance of the dot is <= to the canvas radius, the dot get's a different color.

This creates a circle

JavaScript counts the number of dots that fall within the radius and stores it in a variable

Another variable is used to count the total number of dots.


JavaScript can than use these variables to brutally estimate π
